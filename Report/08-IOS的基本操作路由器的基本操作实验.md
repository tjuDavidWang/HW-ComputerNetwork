# IOS的基本操作路由器的基本操作实验

2023/10/09

## 实验目的

本实验旨在深入理解实验网络的物理组网原理，并掌握其各设备间的连接策略及作用。同时，实验将引导学生学习并实践如何成功登录路由器。最核心的目标是使学生熟悉并掌握路由器操作系统IOS的基本操作，确保在未来的网络配置和管理工作中能够高效地运用这些知识与技能。

## 实验原理 

- 实验网络的物理组网原理主要关注网络中的硬件设备如何通过物理介质（如电缆）连接在一起，以及这些连接如何构建一个功能完整的网络系统。这包括了以下几个方面：

  - 设备角色定义：在一个网络中，不同的设备有其特定的功能和责任，例如路由器负责在不同的网络之间转发数据包，而交换机则在同一个局域网内转发数据帧。

  - 物理介质选择：网络设备之间的连接通常依赖于物理介质，如双绞线、光纤和无线信号。每种介质都有其传输速度、距离限制和环境要求。

  - 拓扑结构：描述了网络中设备的物理或逻辑布局方式。常见的有总线、星形、环形和网状结构等。

- 登录路由器是访问和配置路由器的基础步骤。

  这通常涉及连接到路由器的控制台端口，并使用特定的认证凭据（如用户名和密码）进行登录。

- 路由器的操作系统，即IOS（Internetwork Operating System），是Cisco路由器和交换机的核心软件。它为设备提供了所有的网络功能，如路由、交换、安全和其他网络服务。学习IOS的基本操作是管理和配置Cisco设备的关键，这包括了以下几个方面：

  - 命令行接口（CLI）：IOS的主要操作环境，允许用户输入命令以查询信息或更改配置。

  - 配置模式：IOS中有多种配置模式，如全局配置模式、接口模式等，每种模式都有其特定的命令集，用于对应的配置任务。

  - 文件管理：IOS允许用户管理设备上的文件，如配置文件和IOS映像文件，这通常涉及到文件的保存、备份和恢复等操作。

  - 故障排查工具：IOS提供了一系列的工具和命令，如ping、traceroute和show命令，帮助管理员诊断和解决网络问题。

## 实验设备 

- 操作系统：Windows 10
- 网络环境：局域网
- 软件：Cisco Packet Tracer虚拟实验环境

## 实验步骤

1. 安装并使用Cisco Packet Tracer虚拟实验环境进行模拟练习。

2. 使用不同方法（如：控制台、MODEM、telnet、TFTP）配置IOS。

   - 通过路由器的CONSOLE口，连接至PC的COM口进行本地配置。
   - 利用MODEM连接到aux口，用于远程配置。
   - 通过VTY线路进行telnet。
   - 使用TFTP下载配置文件。
   - 通常，最常用的方法是通过控制台和telnet。

3. 学习IOS的启动过程和不同的配置模式。

   - 设备加电后进行硬件自检。
   - 定位并载入IOS文件。
   - 定位并运行配置文件。完成后，设备可正常工作。

4. 实践在路由器上配置不同的接口模式。

   - 用户模式：主要用于查看路由器状态，有限制的命令。

     直接点击`CLI`便可以进入用户模式

   - 特权模式：用于查看路由器配置和路由信息。

     在用户模式中输入`enable`，切换到特权模式；在特权模式中输入`disable`便可以切换到用户模式

   - 全局配置模式：进行路由器的基本配置。

     在特权模式下输入`configure terminal`便可以进入全局模式

   - 接口模式：针对特定接口进行配置。

     使用如下命令配置路由器上的特定以太网接口，其中`ethernet` 指定你要进入的是以太网类型的接口；`0` 是接口的编号。

     ```
     router(config)# interface ethernet 0
     ```

   - 子接口配置模式：配置特定子接口。

     在全局配置模式下输入命令，便可以进入

     ```
     router(config)#interface fastehernet 0/0.1 
     ```

   - 线路配置模式：配置特定线路，例如console线。

     在特权模式中输入`line console 0` 即切换到线路配置模式

   - 路由配置模式：配置特定的路由协议。

     在特权模式中输入`route rip` 即切换到线路配置模式

5. 此外输入`no shut`可以打开接口，用`end`可以退到特权模式

6. 根据连接基本原则，连接路由器、交换机和PC。

   路由器和PC属于DTE类型，即数据终端设备，交换机和HUB属于DCE类型，即数据通信设备；

   故路由器和交换机、PC和交换机之间用直通线链接，路由器和PC之间用交叉线链接。

## 分析讨论 

- IOS为Cisco的设备提供了强大的配置和管理能力，通过多种模式可以满足不同的配置需求。
- 在网络设备连接中，需要注意设备的类型和使用的线缆种类，以确保正确且高效的连接。
- PT5软件为网络配置提供了模拟环境，使我们能够在不影响实际网络的情况下进行实践。
